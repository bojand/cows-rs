Get a list of cows or a random cow. The `Rust` version. Original [JavaScript Cows](https://github.com/sindresorhus/cows).

## Usage

**(Once published)** Add to your `Cargo.toml`:

```toml
[dependencies]
cows = "1.0"
```

```rust
extern crate cows;

fn main() {
    println!("{}", cows::random());
}
```
